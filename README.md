# StudyManager-console
StudyManager is a console-based Java application developed to practice Object-Oriented Programming by implementing a CRUD system for managing study tasks.



## Features

- Create tasks
- List all tasks
- Search tasks by ID
- Update task name, subject, deadline, and status
- Delete tasks
- Create tasks with or without deadlines
- Handle invalid user input

## Technologies

- Object-Oriented Programming (OOP)
- Java Collections (`ArrayList`, `List`)
- LocalDate API
- Git
